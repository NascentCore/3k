# 算想云微调推理以及评测示例
本项目展示了如何使用算想云平台进行大语言模型的微调训练和评测。主要包含以下内容:

- 使用中文医疗数据集对Gemma-2b模型进行微调
- 基于Rouge评分对微调后的模型进行评测
- 提供完整的训练和评测流程示例代码

项目使用算想云提供的API接口,支持模型训练、部署和推理等功能。评测数据集包含医疗健康领域的问答数据,可用于验证模型在特定领域的表现。

## 运行步骤
1. 安装依赖
```bash
pip install -r requirements.txt
```
2. 配置config.py文件
3. 运行脚本
```bash
python main.py
```
