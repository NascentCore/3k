# 使用说明
使用 `langchain` 接入文心一言。

## 环境准备

首先，创建一个新的 conda 环境，并安装必要的包：

```bash
conda create -n sxwl_patent python=3.9
conda activate sxwl_patent
# pip install openai
pip install 'volcengine-python-sdk[ark]'
pip install langchain
pip install qianfan
```


## 运行
```python
python huajin_feature_extract.py
```
